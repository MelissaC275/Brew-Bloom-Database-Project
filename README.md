# Brew-Bloom-Database-Project
This project was developed as part of an MIS 385 course and focuses on building a database system for a business concept called Brew Bloom.

Tools Used:
- SQL (MySQL)
- Tableau
- Database Modeling (ER Diagram)

Project Overview
- Designed a relational database including zipcode, customer, order, order detail, product, and suppliers
- Created SQL queries and views to analyze:
    - Order totals
    - Revenue per product
    - Customer order activity
    - Inventory stock levels
